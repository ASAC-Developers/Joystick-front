## Getting Started

<!-- Requirements -->
### Requirements

This project uses npm as package manager

<!-- Run Locally -->
### Run Locally

Clone the project

```bash
  git clone git@github.com:ASAC-Developers/Joystick-front.git
```

Go to the project directory

```bash
  cd Joystick-front
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
